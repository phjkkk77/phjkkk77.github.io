⚾ KBO 상대 전적 웹 서비스 (AI 활용 과제 제출용)

이번 트리노드 AI 과제 제출을 위해 초기 코드를 어떻게 수정했는지 비교하기 쉽도록 구조를 분리해 두었습니다.

📂 폴더 구조 안내
수정 전과 후를 명확하게 비교하기 위해 기존 파일에 덮어쓰지 않고 아래와 같이 폴더를 나누어 두었습니다.

kbo/ (원본 폴더): 3개월 전, 강의 과제로 AI 툴을 활용해 태어나서 거의 처음으로 만들어보았던 웹사이트의 원본 코드입니다.

kbo_수정본/ (수정 폴더): 이번 AI 과제 요구사항에 맞춰, 원본 코드에 있던 AI의 얕은 코드와 오류들을 
제가 직접 비판적으로 검토하고 수정한 파일들이 들어있습니다.

📝 수정 및 개선 내용 (필독)
제가 AI의 어떤 코드를 비판적으로 바라보고, 어떻게 개선했는지에 대한 구체적인 내용은 
kbo_수정본 폴더 안에 있는 아래 두 문서에 상세히 정리해 두었습니다.
꼭 참고해 주시면 감사하겠습니다.

[변경사항기록.md]
AI가 존재하지도 않는 데이터를 참조하게 만들거나 텍스트를 대충 하드코딩해 둔 부분을 찾아내어 실제 구조에 맞게 수정한 과정을 적어두었습니다.

데이터의 첫 행만 대충 검사하고 넘어가던 AI의 코드를 뜯어고쳐, 전체 데이터를 꼼꼼히 검사하고 
에러를 3단계로 나누어 보여주도록 개선한 내용이 포함되어 있습니다.

[수정완료현황.md]
현재 1순위로 완료된 핵심 수정 항목(데이터 구조 정정, 검증 로직 강화 등)과 검증 기준을 요약해 둔 문서입니다.
